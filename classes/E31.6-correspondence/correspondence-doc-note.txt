Example transformation for gmn:P70_26_indicates_sender:

INPUT (shortcut):
<letter> gmn:P70_26_indicates_sender <person_john> .

OUTPUT (CIDOC-CRM compliant):
<letter> cidoc:P70_documents <letter/correspondence> .
<letter/correspondence> a cidoc:E7_Activity ;
                       cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
                       cidoc:P14_carried_out_by <person_john> .

Note: The sender is the active party who carries out the correspondence activity 
via P14_carried_out_by. The E7_Activity is automatically typed as AAT 300026877 
(correspondence). Domain is gmn:E31_6_Correspondence.

---

Example transformation for gmn:P70_27_has_address_of_origin:

INPUT (shortcut):
<letter> gmn:P70_27_has_address_of_origin <place_venice> .

OUTPUT (CIDOC-CRM compliant):
<letter> cidoc:P70_documents <letter/correspondence> .
<letter/correspondence> a cidoc:E7_Activity ;
                       cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
                       cidoc:P7_took_place_at <place_venice> .

Note: The origin place is where the correspondence activity took place via 
P7_took_place_at. This models the location from which the letter was written 
and sent. Domain is gmn:E31_6_Correspondence.

---

Example transformation for gmn:P70_28_indicates_recipient:

INPUT (shortcut):
<letter> gmn:P70_28_indicates_recipient <person_maria> .

OUTPUT (CIDOC-CRM compliant):
<letter> cidoc:P70_documents <letter/correspondence> .
<letter/correspondence> a cidoc:E7_Activity ;
                       cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
                       cidoc:P01_has_domain <person_maria> .

Note: The recipient is the target or addressee of the communication via 
P01_has_domain (inverse: P01i_is_domain_of). This creates an asymmetric 
relationship between sender (P14) and recipient (P01), appropriate for 
directed communication. Domain is gmn:E31_6_Correspondence.

---

Example transformation for gmn:P70_29_indicates_holder_of_item:

INPUT (shortcut):
<letter> gmn:P70_29_indicates_holder_of_item <person_carlo> .

OUTPUT (CIDOC-CRM compliant):
<letter> cidoc:P70_documents <letter/correspondence> .
<letter/correspondence> a cidoc:E7_Activity ;
                       cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
                       cidoc:P16_used_specific_object <letter/holding_activity_0> .
<letter/holding_activity_0> a cidoc:E7_Activity ;
                           cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300077993> ;
                           cidoc:P14_carried_out_by <person_carlo> .

Note: The holder is linked through a nested E7_Activity representing the 
holding/carrying activity described in the letter. The correspondence activity 
references this holding activity via P16_used_specific_object. The holder 
carries out the holding activity via P14_carried_out_by. AAT 300077993 
represents "holding (grasping, retaining possession)". Domain is 
gmn:E31_6_Correspondence.

---

Example transformation for gmn:P70_30_refers_to_described_event:

INPUT (shortcut):
<letter> gmn:P70_30_refers_to_described_event <event_cargo_arrival> .

OUTPUT (CIDOC-CRM compliant):
<letter> cidoc:P70_documents <letter/correspondence> .
<letter/correspondence> a cidoc:E7_Activity ;
                       cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
                       cidoc:P16_used_specific_object <event_cargo_arrival> .
<event_cargo_arrival> a cidoc:E5_Event .

Note: Events described in the document are linked via P16_used_specific_object. 
The document's activity references the described event, indicating that the event 
is part of the document's subject matter. Domain is cidoc:E31_Document (not just 
Correspondence), making this property usable with any document type including 
contracts, declarations, and correspondence.

---

Example transformation for gmn:P70_31_has_address_of_destination:

INPUT (shortcut):
<letter> gmn:P70_31_has_address_of_destination <place_cairo> .

OUTPUT (CIDOC-CRM compliant):
<letter> cidoc:P70_documents <letter/correspondence> .
<letter/correspondence> a cidoc:E7_Activity ;
                       cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
                       cidoc:P26_moved_to <place_cairo> .

Note: The destination place is where the letter is sent to via P26_moved_to. 
Together with P70_27_has_address_of_origin (which uses P7_took_place_at), 
this models the spatial transfer of the letter from origin to destination. 
Domain is gmn:E31_6_Correspondence.

Example: Complete Correspondence Transformation

INPUT (shortcut):
<letter_abc123> a gmn:E31_6_Correspondence ;
    gmn:P1_1_has_name "Merchant Letter from Venice to Cairo, March 1450" ;
    gmn:P70_26_indicates_sender <person_john_merchant> ;
    gmn:P70_27_has_address_of_origin <place_venice> ;
    gmn:P70_28_indicates_recipient <person_maria_trader> ;
    gmn:P70_31_has_address_of_destination <place_cairo> ;
    gmn:P70_29_indicates_holder_of_item <person_carlo_porter> ;
    gmn:P70_30_refers_to_described_event <event_cargo_arrival> ;
    gmn:P70_11_documents_referenced_person <person_port_official> ;
    gmn:P94i_2_has_enactment_date "1450-03-15"^^xsd:date .

OUTPUT (CIDOC-CRM compliant):
<letter_abc123> a gmn:E31_6_Correspondence ;
    cidoc:P1_is_identified_by <letter_abc123/appellation/12345678> ;
    cidoc:P70_documents <letter_abc123/correspondence> ;
    cidoc:P67_refers_to <person_port_official> ;
    cidoc:P94i_was_created_by <letter_abc123/creation> .

<letter_abc123/appellation/12345678> a cidoc:E41_Appellation ;
    cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300404650> ;
    cidoc:P190_has_symbolic_content "Merchant Letter from Venice to Cairo, March 1450" .

<letter_abc123/correspondence> a cidoc:E7_Activity ;
    cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300026877> ;
    cidoc:P14_carried_out_by <person_john_merchant> ;
    cidoc:P7_took_place_at <place_venice> ;
    cidoc:P26_moved_to <place_cairo> ;
    cidoc:P01_has_domain <person_maria_trader> ;
    cidoc:P16_used_specific_object <letter_abc123/holding_activity_0> ,
                                   <event_cargo_arrival> .

<letter_abc123/holding_activity_0> a cidoc:E7_Activity ;
    cidoc:P2_has_type <http://vocab.getty.edu/page/aat/300077993> ;
    cidoc:P14_carried_out_by <person_carlo_porter> .

<letter_abc123/creation> a cidoc:E65_Creation ;
    cidoc:P4_has_time-span <letter_abc123/creation/timespan> .

<letter_abc123/creation/timespan> a cidoc:E52_Time-Span ;
    cidoc:P82_at_some_time_within "1450-03-15"^^xsd:date .

Note: This complete example shows how all correspondence properties share the 
same E7_Activity node (letter_abc123/correspondence), creating a unified model 
of the correspondence event. Persons mentioned in the letter content use 
P70_11_documents_referenced_person and link directly via P67_refers_to. The 
holding activity and described event are nested within the correspondence 
activity via P16_used_specific_object.
